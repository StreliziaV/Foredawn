About the page numbers of the report, the main body of the report is 30 pages, and there are three more pages for report cover, content lists, and references.

If you want to run the program:
	1. run Fordawn.sql to build database.
	you may need the two lines of code, too:
	alter user 'root'@'localhost' identified with mysql_native_password by 'you password';
	flush privileges;
	2.  cmd type: npm install
	3. cmd type: node server.js to run the server
	4. now you can enter any of the html page to test the functions.
